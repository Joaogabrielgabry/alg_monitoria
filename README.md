# 📘 Exercícios em C - Monitoria

Repositório dedicado ao armazenamento de exercícios desenvolvidos em linguagem C durante sessões de monitoria.

## 🎯 Objetivo

Este projeto tem como objetivo:

* Praticar e reforçar conceitos fundamentais da linguagem C
* Auxiliar alunos no aprendizado de lógica de programação
* Servir como material de apoio para estudos futuros
* Documentar a evolução dos exercícios realizados em monitoria

## 🧠 Conteúdos abordados

Os exercícios cobrem temas como:

* Estruturas condicionais (`if`, `else`, `switch`)
* Estruturas de repetição (`for`, `while`, `do while`)
* Vetores e matrizes
* Funções
* Ponteiros
* Manipulação de strings
* Estruturas (`struct`)
* Algoritmos básicos

## ▶️ Como executar

Para compilar e executar os programas, utilize um compilador como o `gcc`:

```bash
gcc nome_do_arquivo.c -o programa
./programa
```

## 👨‍🏫 Contexto

Os exercícios foram desenvolvidos durante monitorias acadêmicas, com foco no apoio a alunos iniciantes na programação.
